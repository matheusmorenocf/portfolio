IP Address Tracker

HTML CSS Javascript API

Desafio Frontend Mentor.
O desafio consiste em criar um IP Address Tracker, é uma aplicação web que recebe um endereço IP ou domínio como entrada e retorna informações sobre a região e localização correspondente em um mapa. O projeto é educacional e tem como objetivo exercitar o uso de APIs, incluindo a API de geolocalização de IP da IPify e a biblioteca LeafletJS para gerar o mapa.
